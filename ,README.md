# Simple Omelette Recipe Page

## Project Overview

This project is a responsive recipe webpage built using **HTML** and **CSS**. It presents a simple omelette recipe in a clean, modern, and easy-to-read layout while demonstrating the use of semantic HTML and modern CSS styling techniques.

## Features

- Responsive recipe page layout
- Semantic HTML structure
- Hero recipe image
- Preparation time section
- Ingredients list
- Step-by-step cooking instructions
- Nutrition information table
- Clean typography and color scheme
- Mobile-friendly design
